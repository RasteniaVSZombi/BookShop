# 📚 BookShop

Приложение «Книжный магазин», разработанное в рамках лабораторной работы №4 по дисциплине «Основы программирования».

## 🧾 Описание проекта

Данное приложение моделирует работу книжного магазина.
Пользователь может создавать книги, хранить их по жанрам, выполнять поиск и продавать их.

Проект реализован с использованием технологии Windows Forms и принципов объектно-ориентированного программирования.



## ⚙️ Используемые технологии

В проекте были использованы следующие средства:

* Платформа: **.NET 8.0**
* Язык программирования: **C#**
* Среда разработки: **Visual Studio 2022 и новее**
* Типы проектов:

  * Приложение **Windows Forms (Microsoft)**
  * **Библиотека классов (Microsoft)**
  * **Тестовый проект MSTest (Microsoft)**



## 📦 Структура проекта

Проект состоит из нескольких частей:

* **BookShop (UI)**
  Пользовательский интерфейс приложения (Windows Forms)

* **BookLibrary**
  Основная логика приложения (классы Book, BookShelf, Shop)

* **BookShop.Tests**
  Модульные тесты для проверки работы программы



## 🚀 Основные возможности

* создание книги вручную;
* генерация случайных книг;
* хранение книг по жанрам;
* поиск книги по названию и ID;
* продажа книг;
* отображение баланса магазина.



## 🧠 Особенности реализации

* используется объектно-ориентированный подход;
* поддержка нескольких шкафов одного жанра;
* гибкая система хранения книг;
* разделение логики, интерфейса и тестов;
* обработка ошибок (try-catch).



## 🧪 Тестирование

В проекте реализованы модульные тесты с использованием MSTest.

Проверяются:

* создание книги;
* добавление книги в шкаф;
* поиск книги;
* продажа книги;
* корректность обновления баланса.



## ▶️ Запуск проекта

1. Открыть решение (.sln) в Visual Studio
2. Установить проект **BookShop** как запускаемый
3. Нажать **Start (F5)**



## ⚠️ Примечание

Для корректной работы проекта требуется установленная платформа .NET 8.0 и Visual Studio 2022 или новее.
