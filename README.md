# BookShop

Простое приложение «Книжный магазин», разработанное на C# (.NET, Windows Forms).
Позволяет создавать книги, хранить их в шкафах по жанрам, искать и продавать их.

Проект выполнен в рамках лабораторной работы по дисциплине «Основы программирования». Лабораторная Работа 3

Возможности:
- Создание новой книги
- Случайная генерация книги
- Хранение книг по жанрам в шкафах
- Поиск книги по названию или ID
- Продажа книги
- Отображение баланса магазина

Структура проекта:
BookShop:
- BookShop.UI        # Windows Forms интерфейс
- BookShop.Library   # основная логика приложения
- BookShop.Tests     # unit-тесты

Основные классы:

а) Book – описание книги:
- название
- автор
- ID
- жанр
- страницы
- цена

б) BookShelf – книжный шкаф:
- хранит книги одного жанра
- имеет ограниченную вместимость

в) Shop – магазин:
- содержит шкафы
- хранит баланс магазина

Запуск:
1) Открыть решение BookShop.sln в Visual Studio
2) Запустить проект BookShop.UI
